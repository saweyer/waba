About the Waba Source Distribution

Welcome to the Waba Source Distribution! To compile the C++ code properly, you should stick this folder inside the NCT_Projects:Demos_NCT folder of the NewtonDev distribution. Just make sure that the MPW C++ compiler is compiling with No Debug, and set to the Demos_NCT:Waba package.

This main directory contains the source code for the Waba Virtual Machine. Other stuff is located in three specific subdirectories:

icons
Various icon files for the VM
docs
Your best friend. All sorts of on-line documentation for getting Waba apps and the VM compiling and running.
aux
Additional stuff: class libraries, simple waba apps, auxillary useful things.

For more documentation, including an intro to the VM, see the online docs at the Waba for Newton Homepage.
